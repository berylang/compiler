# Security Policy

## Supported Versions

Currently, the Bery compiler is in active development.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.0   | :x: |

## Reporting a Vulnerability
If you discover a security vulnerability within the Bery compiler (e.g., buffer overflows in the lexer, AST memory leaks, or malicious LLVM IR generation exploits), please do not open a public issue. 

Instead, report it directly to the project lead or the core Bery team members. We will attempt to acknowledge your report within 48 hours and provide a timeline for a patch.
