# Contributing to Bery

Welcome to the Bery programming language project! Bery is a compiled, statically typed language with an LLVM backend. This project is actively developed by the Bery team.

## Core Project Rules
We adhere to strict development guidelines. By contributing to this repository, you agree to the following constraints:
* **No AI Assistance:** No AI tools are allowed in any project work or code generation.
* **Transparency:** All communication and votes must be transparent.
* **Definition of Done:** No task can be marked complete without a working implementation and documented output.

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature (`git checkout -b feature/amazing-feature`).
3. Ensure your C++ code complies with the existing AST and CodeGen.
4. Test your changes against the `bery_out.ll` LLVM IR generation.
5. Commit your changes (`git commit -m 'Added: amazing feature'`).
6. Push to the branch (`git push origin feature/amazing-feature`).
7. Open a Pull Request using the provided template.
